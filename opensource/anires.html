1.1. 使用方法

git clone https://gitee.com/zhousiwei/anires.git
cd anires
npm install
npm run dev

1.2. 部署方法

  1. 编辑 config/index.js，修改第 10 行的 assetsPublicPath，值为 项目名/public。如果你没有修改项目名 anires，则可跳过此步骤。

  2. 编译、上传

    npm run build
git add .
git commit -m "update"
git push

  3. 开启 GitHub Pages 功能，把生成的目标文件public/*放在你的web服务器上。

  4. 访问地址：https://your-github-username.github.io/anires/public
